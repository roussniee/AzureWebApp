F-D Comparison File Upload

Please note that you can only upload one F file and one D file at a time. When you upload a F file along with a D file, it will generate one corresponding output file, which can be downloaded from the View Files page. Please refer to the date and time in the title name to determine which file is your output file.
If you have multiple D files that need to be compared to F data, you will need to upload the F file and each D file and run the comparison separately for each one.
